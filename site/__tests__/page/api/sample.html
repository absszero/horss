宜蘭小子「暫停哲學」奪世界撞球冠軍！柯秉逸：它教了我人生的一切

2023.06.28

撰文者：簡永昌

商業周刊第1859期

  • +A-A
 

「難道又要停在這裡了嗎？」柯秉逸擦著球桿、一面想。這是2023年世界花式撞球大師賽的4強賽，他第6次參加這場世界撞球界的指標賽事，但之前5次都沒能突破4強賽。

心魔在他心底蔓延：一方面想到自己總在這關敗下陣來；另一個聲音卻告訴自己從沒輸過眼前這個對手，球賽在他的技術與心理狀態搖擺間進行，導致他打得綁手綁腳，居於劣勢。

小檔案＿柯秉逸

出生：1989年
學歷：台北市立復興高中、台北海洋科技大學
榮譽：
・2023年世界9號球大師賽冠軍
・2022年美洲豹波多黎各男子10號球公開賽季軍
・2022年APF亞洲9號球公開賽冠軍

他喊了暫停，深吸一口氣。「不要再想『不該輸』，該專注在眼前這顆球。」他正念思考，慢慢用自己的步伐追回比數，以11比9險勝，搶到總決賽門票。

冠亞軍戰，他更毫無包袱、放手一搏。最後贏得冠軍時，他跳上球檯握拳歡呼，全場報以熱烈掌聲，柯秉逸也成為30年來，台灣首位在這項頂級賽事奪冠的選手。

小辭典＿世界花式撞球大師賽

World Pool Masters，是撞球界極具有指標意義的比賽，自1993年起開始每年舉辦一次。

採邀請制，由大會邀請全球16至24位頂尖選手參賽，2023年在英國舉辦，冠軍獎金為4萬美元，柯秉逸為台灣唯一受邀的選手，並獲得冠軍。

之前台灣選手的最佳成績為2003年夏揮凱獲得的亞軍。

愛因斯坦曾說：「一場撞球，有600萬種打法。」不像短跑選手靠著爆發力的衝刺、體操選手策略性以不同難度的動作獲取高分，撞球不只靠體力，更是心理素質的持久戰與智力賽。

虎爸帶他著正裝四處學球
「不讓人覺得我們只是玩玩」

撞球比的是什麼？第一是技術，透過讓母球與子球、球檯邊精確碰撞，並臨場反應調整球路，其次更是心理素質，有可能因為對手連續得分，連一次上場機會都沒有，光是等待都是種壓力，更不能因為自己未上場而分心。

「一定要往下想2至3步，」柯秉逸說，每次發球，力道該如何拿捏才能讓母球停在最佳位置？「甚至我也必須預想最壞的可能，」如果這一球已無希望，又該如何做一個燙手山芋的球給對手？下棋般的多套布局計畫很重要。

場上的一秒鐘，全來自於場下10年功。柯秉逸今年34歲，打撞球的歲月卻已超過20年。父親柯志明是台灣乙組國家撞球運動員，在宜蘭開設撞球館，柯秉逸從小在球館裡玩，小學時的桿上功夫已能打敗一些成人常客。

「我爸把球場環境控管得比較單純，所以我從來不覺得撞球館環境有多差。」他說，國小6年級他跟父親說想打撞球，父親問他：「認真的嗎？」他點了頭，從此父親像變了一個人。

「我們家本來是媽媽扮黑臉、爸爸當白臉，從那天起角色忽然互換，爸爸突然變得很兇，我若沒打好，他就一直罵。」他回憶，父親除了陪練、教球外，有空便開車帶兒子全省跑，向各地選手討教學技術。

父子倆常常一出門就是一週，為了省錢，經常直接以車為家，早上醒來在超商買礦泉水洗把臉就去學球，父親要求兒子以英式規範，穿上西裝背心打領帶，兒子當時覺得很怪，父親卻說：「這樣才不會讓別人覺得我們只是玩玩，是真的想要把球打好。」

「我小時候很怕爸爸，除了兇，他還會想到很多奇怪方法提升我的專注力。」柯志明認為專注力是撞球選手最重要的能力，他會帶兒子到中正紀念堂和憲兵對看，或把球館大門拉下，把音樂開到最大聲叫他練球，希望訓練柯秉逸能完全不受外在因素影響，全神貫注打好每顆球。

打好每一顆球，不到最後絕不放棄，柯秉逸說：「只要對手沒有把最後那顆9號球打進袋，我都覺得我還有機會。」

不像天才弟，他扎實練功
一套正向思考打敗緊張失誤

他往往能演出大逆轉。如18歲參加泰國公開賽，冠軍賽對上世界排名第一的菲律賓選手，15局只差一局就要被對方拿下，他卻奇蹟式的連拿11局大逆轉，奪下冠軍。而這回在世界花式撞球大師賽上，從失分到一路追平，甚至還能打出一桿進洞。

「他不是天才，但是不屈不撓，適合打耐力賽。」柯秉逸在復興高中的教練陳冠列形容他，弟弟柯秉中說：「我哥穩紮穩打，他學來的一切都不容易，但是我有不懂的問他，他從來不藏私。」

小他6歲的弟弟柯秉中是天才型選手，第一次出國比賽就連奪兩場國際賽亞軍、15歲跳進台灣職業撞球界，成為史上最年輕的冠亞軍賽選手。但哥哥柯秉逸卻是一步一腳印、沒有跳過任何級，如同龜兔賽跑中的烏龜，穩穩當當走向目標。

慢，卻扎實，在高壓賽場上，柯秉逸有套「正向思考法」：「如果一直告訴自己『不要失誤』、『不要緊張』，腦袋裡就一直出現『失誤』『緊張』這些負面詞。」這時他會把腦袋清空，給自己「專心」、「進洞」等正面字彙：「這種正向能量，我覺得非常有用。」

「越緊張，越容易失誤，」看著對手持續進攻入袋，情緒波動會影響臨場表現，柯秉逸會適時喊出暫停，阻斷對方進攻氣勢，為自己爭取空間喘息，調整好步伐再戰。


疫情停擺期想起69歲偶像
「路那麼長，不必只看眼前」

然而長達3年的疫情，是上天喊的暫停。相較於歐洲對手們只停賽1年，困在台灣、長達近3年無法出國比賽的柯秉逸跟柯秉中，積分幾乎歸零，影響了兩人的世界排名，也曾一度有「練了要幹嘛？又不能出國比賽」的心酸。

在最困窘的時候，他想起自己的偶像。現年69歲、人稱「撞球魔術師」的菲律賓籍撞球選手雷耶斯（Efren Manalang Reyes），他最為人稱道的不僅是球技，而是待人。

「他到現在還是一線選手，我曾在賽場上遇過他，我年紀不到他一半，他也不會看不起我，笑咪咪教我這顆球要怎麼打，那個關要怎麼過。」柯秉逸說：「他讓我知道，撞球選手生涯很長，只要狀況好、打到70歲都沒有問題。路還那麼長，不必只看眼前。」

「如果這是我要做一輩子的事，為什麼要急？」暫停，或許是上天給的禮物，要柯秉逸再次調整心態。恰巧小弟柯秉漢一直想經營社群，三兄弟決定認真拍影片講解撞球術語跟技巧，同時維持手感，也才讓他回歸賽場後，半年內就成功奪冠。

或許人生如一場撞球賽，排名、獎金、勝敗常常擾亂你的心神，但你需要的是，深吸一口氣，將心力專注在眼前的這顆球；光想下一步還不夠，必須想好下兩步再動手；當對手氣勢影響到自己，你可以技巧性喊暫停，讓戰情洗牌。

「有些人以為撞球就只是球碰球而已，但不只這樣，它教了我關於人生的一切。」柯秉逸說。在對手最後一顆9號球進袋之前，都不要放棄。以自己的步伐去走，無論世間認為是快或慢，不汲汲於迅速成功，他等待的，終究是人生的長尾效應。
